Privacy Policy

EVIT Smart Charger Alexa Skill

Last updated: 30 Jan,2026

Introduction

This Privacy Policy explains how EVIT (“we”, “our”, “us”) collects and uses information when you use the EVIT Smart Charger Alexa Skill.

Data We Collect

To provide the skill’s functionality, we may access the following information:

  • EVIT account identifier used for account linking
  • EVIT charger details such as charger name, status, and charging state
  • User commands such as start or stop charging

Data We Do NOT Collect

  • Alexa voice recordings
  • Personal Alexa profile information (name, email, phone number)
  • Payment or billing information

How We Use Data

Collected data is used solely to:

  • Authenticate EVIT users
  • Identify chargers linked to the user
  • Execute user-requested charger actions
  • Provide spoken responses via Alexa

Data Sharing

We do not sell, rent, or share user data with third parties. Data is used only to operate EVIT charger services.

Data Retention

Authentication tokens are stored securely and expire automatically. Charger data is retained only as required to provide the service.

Security

We use industry-standard security measures, including encrypted communication and access control, to protect user data.

User Control

You can disable the EVIT Smart Charger skill at any time through the Alexa app, which immediately stops all access.
